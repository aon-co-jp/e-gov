# e-gov.info (한국어)

(1) 디지털 정부(무서류 행정 신청)와 (2) 개인부터 무역상사까지 대응하는
온라인 무역/부동산 플랫폼을, LINE 앱・웹사이트・편의점 멀티미디어
단말기 등 여러 창구에서 이용할 수 있도록 통합하는 프로젝트입니다.

> ⚠️ **본 프로젝트는 아직 샘플/데모입니다. 실제 서비스가 아닙니다.**
> 웹사이트를 이용하든 LINE 계정을 친구로 추가하든 매번 이 안내가
> 표시됩니다. 전자공증 및 법적 구속력이 있는 전자계약(부동산 매매・
> 임대차 계약 체결)은 정식 승인이 내려질 때까지 현재로서는 구현하지
> 않습니다.

## 두 개의 축

### 1. 디지털 정부(e-Government)

- **다양한 창구**: LINE 앱・웹사이트 외에도, 웹사이트나 스마트폰 조작이
  익숙하지 않은 분들을 위해 편의점 멀티미디어 단말기(Loppi・Fami Port・
  세븐일레븐 복합기 등)를 신청 창구로 사용할 수 있게 합니다.
- **단계별 본인확인**: 거래 금액・중요도에 따라 인증 강도를 달리합니다
  (발신자번호표시+콜백 확인 → 이메일 OTP → 마이넘버카드의 스마트폰
  NFC 스캔).
- 무서류 온라인 신청과, 여러 부처・지자체를 아우르는 원스톱화를
  지향합니다.

### 2. 온라인 무역 플랫폼(AI 챗커머스)

- 식품부터 자동차・오디오 기기까지, 폭넓은 상품을 LINE・웹 양쪽에서
  AI와의 대화로 발견・주문할 수 있는 종합 쇼핑몰입니다.
- 개인부터 무역상사까지 대응하는 마켓플레이스형. 기존 대형 플랫폼보다
  수수료를 대폭 낮춥니다.
- **AI 신용조사・외상매입・매출채권 보증**: AI가 산출한 신용점수에 따른
  후불 매입, 전자세금계산서 중복 조사, 매출채권 보증 기능.
- **부동산 투자・AI 건축사무소**: 검색한 토지 정보를 바탕으로 AI가
  평면도를 제안. 투기 자금의 과도한 유입을 조장하지 않는다는 설계
  원칙.
- 초기 단계에서는 실제 재고 매입 없이, 애드센스 수익을 통한 샘플
  운영으로 AI 챗커머스 UX를 검증합니다.

## 참고한 해외 사례

- 에스토니아 e-Estonia / X-Road(분산형 데이터 연계 기반)
- 아제르바이잔 ASAN xidmet(원스톱 서비스 센터・이동 서비스)

## 자동 조사・자동 마케팅 기능

디지털 정부 관련 주제에 대해 일본어・영어로 정기적으로 자동 조사
(GitHub Search API를 통한 실제 데이터 취득, Google 검색 링크 자동
생성)를 실시하고, 각 기능의 홍보 문안도 정기적으로 자동 생성합니다
(`/research` 페이지 참조).

## LINE 연동

웹사이트를 둘러보는 것 외에도, LINE 공식계정을 친구로 추가하여 채팅으로
문의할 수 있습니다(키워드 기반 규칙형 AI 챗커머스 응답, 예: "신청하고
싶다", "구매하고 싶다", "외상매입", "토지 검색" 등). 친구 추가용 QR
코드는 LINE 공식계정의 Basic ID가 발급된 후 게재할 예정입니다.
